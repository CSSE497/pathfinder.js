Class: Commodity

Commodity

new Commodity(id, start, destination, route)

Represents a commodity in the Cluster/Vehicle/Commodity (CVC) model. The functions in this class provide notifications about this commodity This class should never directly be instantiated. Instead, it will be created automatically and passed into the callback of functions such as commodityWasRequested()
